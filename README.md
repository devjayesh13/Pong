# Pong
This project is me dipping my toes in some actual game development.
I used raylib for rendering everything and c++ for other stuff for the game.
The project is just a remake of pong from atari 2600, it is relatively simple game thats why I made a remake.

I might give it some updates in the future, like I might add particles and bloom effect using shaders, 
but its time consuming so I will do it later.

And also I know my code is not optimized and I will also fix that later.

This repository is just to look at my code.
